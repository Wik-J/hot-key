# 使用快捷键

## 显示/隐藏资源管理器
- command + b

## 使用live server
- command + l command + o
- command + l command + c

## 预览md
- command + k + v
- command + shift + v

## 展示插件
- command + shift + x

## 显示当前资源所在文件夹
- command + k + r

## 复制当前资源的路径
- command + k + p

## 提醒
- command + k 是vscode的组合键，按完松开后再按其他键才会生效
