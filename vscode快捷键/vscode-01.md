# 操作文件

## 切到files explorer区域
- vim —— ctrl + shift + h
- vscode —— ctrl + ;

## 切到editor区域
- vim —— ctrl + shift + l
- vscode —— ctrl + '

## 移动光标
- j k

## 折叠/展开
- h l

## 创建文件
- a —— 在files explorer区域用
- <Leader> + n + f —— 在editor区域用

## 创建文件夹
- A —— 在files explorer区域用
- <Leader> + n + d —— 在editor区域用

## 重命名
- r

## 删除
- d
