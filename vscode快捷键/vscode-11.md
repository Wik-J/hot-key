# debug

## start
- F5

## stop
- shift + F5

## restart
- shift + cmd + 0
- workbench.action.debug.restart

## 打断点
- shift + cmd + 9
- editor.debug.action.toggleBreakpoint

## step over(单步跳过)
- cmd + '
- workbench.action.debug.stepOver

## step into
- shift + command + '
- workbench.action.debug.stepInto

## step out
- F5
